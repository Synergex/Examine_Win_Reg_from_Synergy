%RegDeleteValue

Description

 Delete a value from the Windows Registry

Interface

 
status = %RegDeleteValue(reghdl ,hive ,key ,value [,flush])
    or
xcall RegDeleteValue(reghdl ,hive ,key ,value [,flush])

Arguments

 reghdl A registry session handle returned by RegStartSession. (i4)
hive The Registry hive in which the key that you wish to create resides.  This can be one of the following (as defined in PSG:registry.def:
HKEY_CLASSES_ROOT
HKEY_CURRENT_CONFIG
HKEY_CURRENT_USER
HKEY_LOCAL_MACHINE
HKEY_USERS
HKEY_PERFORMANCE_DATA (Windows NT only)
HKEY_DYN_DATA         (Windows 95 and Windows 98 only)
key The registry key (directory) from which the value should be deleted. (a)
value The name of the value to be deleted from the specified key in the specified hive. (a)
flush An optional flag specifying that the value deletion should be flushed to disk immediately.  By default, the registry is flushed to disk only periodically (as determined by the operating system) and at system shutdown.  To cause the deletion to be flushed to disk immediately, pass REG_FLUSH. (n)

Return Value

 If successful the routine returns REG_SUCCESS (0).  Otherwise one of the REG_ERR_ error states is returned (as defined in PSG:registry.def).

Remarks

 This routine deletes a value from a key in the Registry.

Related Routines

 RegDeleteKey, RegSetBinary, RegSetDword, RegSetString

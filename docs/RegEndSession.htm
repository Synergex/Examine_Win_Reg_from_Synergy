%RegEndSession

Description

 End a Windows Registry session

Interface

 
status = %RegEndSession(reghdl)
    or
xcall RegEndSession(reghdl)

Arguments

 reghdl The handle of the open DLL which was obtained from RegStartSession. (i4)

Return Value

 This routine always returns REG_SUCCESS.

Remarks

 This routine ends a Registry session by closing the DLL which was opened in the RegStartSession routine. It should be the last routine called in the Registry session.

RegEndSession must be the last routine called in the Registry session.

Related Routines

 RegStartSession

Example

 
.include "PSG:psg.def"
.align
record
    regsess     ,i4
.proc

    .
    .
    .

    xcall RegEndSession(regsess)

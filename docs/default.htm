Windows Registry Routines

These routines allow a Synergy application to retrieve data from and store data to the Windows Registry.

Routine Description
RegCreateKey Create a new key in the Windows Registry
RegDeleteKey Delete a key from the Windows Registry
RegDeleteValue Delete a value from the Windows Registry
RegEndSession End a Windows Registry session
RegGetBinary Get a binary (REG_BINARY) value from the Windows Registry
RegGetDword Get an integer (REG_DWORD) value from the Windows Registry
RegGetString Get a String (REG_SZ) value from the Windows Registry
RegSetBinary Set a binary (REG_BINARY) value in the Windows Registry
RegSetDword Set an integer (REG_DWORD) value in the Windows Registry
RegSetString Set a String (REG_SZ) value in the Windows Registry
RegStartSession Start a Windows Registry session
